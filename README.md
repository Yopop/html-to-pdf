## Requirements

- HtmlToPdf depends on PHP 5.4+
- Shell functions must be enabled
- Unix based operating system

## Installation

Add ``spiritix/html-to-pdf`` as a require dependency in your ``composer.json`` file:

    composer require spiritix/html-to-pdf
